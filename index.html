Graph isomorphism checking tool

Welcome to the Graph isomorphism checking tool, the tool we created based on the result of our homework about graphs
  • The left box and the right box are the canvas for graph 1 and graph 2, respectively.
  • You can create vertices by left click and drag.
  • You can create edges by right click and drag from one node to another.
  • After finish sketching the graph, you can click the "Check isomorphism" button to see the result and the explaination.
  • There currently no "reset" button so if you want to check another graphs, you need to refresh the page.
